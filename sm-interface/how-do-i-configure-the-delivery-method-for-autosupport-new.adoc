= 如何配置AutoSupport 的交付方法？
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
要访问AutoSupport 交付方法的配置任务、请转到菜单：Support[支持中心]、然后单击* AutoSupport *选项卡。

支持以下协议：HTTPS、HTTP和SMTP。
