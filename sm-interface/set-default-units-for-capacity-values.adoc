= 在 SANtricity System Manager 中设置容量值的默认单位
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SANtricity系统管理器可以吉字节(GiB)或兆字节(TiB)为单位显示容量值。

首选项存储在浏览器的本地存储中、因此所有用户都可以拥有自己的设置。

.步骤
. 单击窗口右上角的 *User Settings* 图标。
. 在显示容量值下拉列表中，选择 *Gibibytes* 或 *Tebibytes*。
+
有关缩写词和值、请参见下表。

+
[cols="1a,1a"]
|===
| 缩写 | 价值 


 a| 
GIB
 a| 
1、024^3^字节



 a| 
TIB
 a| 
1、024^4^字节

|===
