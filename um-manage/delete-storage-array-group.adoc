= 删除存储阵列组
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以删除不再需要的一个或多个存储阵列组。

.关于此任务
此操作仅删除存储阵列组。与已删除组关联的存储阵列仍可通过全部管理视图或与其关联的任何其他组进行访问。

.步骤
. 从管理页面中、选择菜单：管理组[删除存储阵列组]。
. 选择要删除的一个或多个存储阵列组。
. 单击 * 删除 * 。
