= 查看AutoSupport 状态
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
AutoSupport 页面将显示AutoSupport 功能和各个AutoSupport 功能当前是否已启用。

.步骤
. 选择菜单：Support[支持中心> AutoSupport ]选项卡。
. 查看页面右侧选项卡下方、查看是否已启用基本AutoSupport 功能。
. 将光标悬停在问号上方可查看是否已启用各个AutoSupport 功能。
