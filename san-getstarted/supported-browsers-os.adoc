= SANtricity 软件浏览器和操作系统支持
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SANtricity 软件支持多种类型的浏览器和操作系统。



== 浏览器

支持以下浏览器和版本。

[cols="1a,1a"]
|===
| 浏览器 | 最低版本 


 a| 
Google Chrome
 a| 
81.



 a| 
Mozilla Firefox
 a| 
80



 a| 
Safari
 a| 
14.



 a| 
Microsoft Edge
 a| 
90

|===
[NOTE]
====
对于Unified Manager、必须安装Web服务代理并可供浏览器使用。有关详细信息，请参见 https://docs.netapp.com/us-en/e-series/web-services-proxy/index.html["SANtricity Web 服务代理概述"^]

====
