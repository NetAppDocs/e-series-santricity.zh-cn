"性能"页面提供了监控存储阵列性能的简单方法。



== 我可以从性能数据中了解哪些内容？

性能图形和表几乎实时显示性能数据、这有助于您确定存储阵列是否遇到问题。您还可以保存性能数据以构建存储阵列的历史视图、并确定问题发生的时间或导致问题的原因。
