= 创建工作负载
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以为任何类型的应用程序创建工作负载。

.关于此任务
工作负载是指支持应用程序的存储对象。您可以为每个应用程序定义一个或多个工作负载或实例。

.步骤
. 选择菜单： Storage[Volumes] 。
. 选择菜单：创建[工作负载]。
+
此时将显示创建应用程序工作负载对话框。

. 使用下拉列表选择要为其创建工作负载的应用程序类型、然后键入工作负载名称。
. 单击 * 创建 * 。


.完成后
您已准备好向创建的工作负载添加存储容量。使用*创建卷*选项为应用程序创建一个或多个卷、并为每个卷分配特定容量。
