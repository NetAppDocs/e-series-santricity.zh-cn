= Snapshot卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果要读取或写入快照数据、您可以创建快照卷并将其分配给主机。快照卷与基础卷具有相同的特征(RAID级别、I/O特性等)。

创建快照卷时、您可以将其指定为_read-onl_y或_read-write accessible_。

创建只读Snapshot卷时、不需要添加预留容量。创建读写Snapshot卷时、必须添加预留容量以提供写入访问。
