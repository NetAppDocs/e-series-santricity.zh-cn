= 删除远程存储对象
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
导入完成后，如果您不再需要在本地设备和远程设备之间复制数据，则可以删除远程存储对象。

确保没有与要删除的远程存储对象关联的导入。

删除远程存储对象后、本地设备与远程设备之间的连接将被删除。

.步骤
. 选择菜单：Storage[远程存储]。
. 从列表中、选择要删除的远程存储对象。
. 单击 * 删除 * 。
+
此时将显示确认删除远程存储连接对话框。

. 键入 `remove` 然后单击 * 删除 * ，确认操作。
+
此时将删除选定的远程存储对象。
