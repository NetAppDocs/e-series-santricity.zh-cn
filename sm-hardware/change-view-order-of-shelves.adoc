= 更改 SANtricity System Manager 中架子的视图顺序
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以更改硬件页面上显示的磁盘架顺序、以匹配机柜中磁盘架的物理顺序。

.步骤
. 在左侧面板中，选择 *Hardware*。
. 如果图中显示了驱动器，请单击*控制器和组件*选项卡。
+
此图将发生变化，以显示控制器，而不是驱动器。

. 从磁盘架视图的右上角、选择向上或向下箭头以重新排列硬件页面上显示的磁盘架顺序。
